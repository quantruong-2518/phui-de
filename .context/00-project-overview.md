# PHỦ ĐÊ - Project Overview

## Mục đích

All-in-one platform cho bóng đá phủi: quản lý đội, đặt sân, ăn uống, mua sắm

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase (PostgreSQL)

## Core Features

1. **Quản lý đội bóng & Thống kê**
   - CRUD cầu thủ
   - CRUD trận đấu
   - Stats & leaderboard
   - Live scoring

2. **Đặt sân**
   - Tìm sân gần nhất
   - Booking calendar
   - Thanh toán online

3. **Ăn uống**
   - Tìm quán quanh sân
   - Order & ship đến sân

4. **Mua sắm**
   - Shop đồ bóng đá
   - Ship tận sân
